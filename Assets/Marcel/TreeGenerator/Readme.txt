Procedural Tree Generator
By Marcel Terblanche
phatphrog@gmail.com

-Click [Random Tree] to generate a tree based on a random seed and random tree parameters.

-Click [Random Seed] to generate a tree based on a random seed with the currently selected tree parameters.

-Click and drag anywhere onscreen to rotate the camera.

-Zoom in and out using the mouse wheel.

-Adjust tree parameters using the sliders.

-Click [Quit] to quit the application.
